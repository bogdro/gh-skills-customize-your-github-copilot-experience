
# 📘 Assignment: Hangman Game

## 🎯 Objective

Practice Python fundamentals by building a classic Hangman word‑guessing game. This project reinforces string manipulation, loops, conditionals, and user input handling while giving you a fun, interactive program.

## 📝 Tasks

### 🛠️ Core Hangman Game

#### Description
Write a Python program that lets a player guess letters to uncover a hidden word. The program should manage the game loop, track guessed letters, and display progress after each guess.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list.
- Prompt the player for one letter at a time.
- Display the current word state using blanks and revealed letters (e.g. `_ a _ _ m a n`).
- Keep track of incorrect guesses and limit the number of attempts.
- End the game with a win message if the word is guessed, or a lose message if attempts run out.

### 🛠️ Enhancements (Optional)

#### Description
Once the core game is working, add one or more extra features to make the experience richer.

#### Requirements
Your enhanced version may include any of the following:

- Load the word list from an external file instead of hard‑coding it.
- Allow the player to play multiple rounds without restarting the program.
- Display letters that have already been guessed.
- Give the player a hint or category for the word.

Feel free to invent your own enhancements as long as they build on the core game functionality.
